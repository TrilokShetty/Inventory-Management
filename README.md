# Inventory Management System

A simple inventory management system built with Node.js, Express, EJS, and MongoDB.

## Installation

1. Clone the repository:
```sh
git clone https://github.com/TrilokShetty/Inventory-Management.git

```

2. Install dependencies:
```sh
npm install
```

3. Set up MongoDB:
   - Ensure MongoDB is running locally or use a cloud MongoDB connection.
   - Update the connection string in `app.js` if needed.

4. Start the application:
```sh
node app.js
```

## Viewing the Application

1. Open your browser and go to:
```sh
http://localhost:3000
```

That's it! You're ready to use the Inventory Management System.
